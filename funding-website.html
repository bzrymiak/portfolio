01.

Project Overview

Sustainable Project Funding Opportunities is a subsection of Embark Sustainability's larger website. Using 6 rows of data from a large funding/award/program application spreadsheet, my team organized this information to create an easily-navigated resource site. The full site can be viewed here!

Low-fidelity sketches for the grant information page layout.

02.

Objective

Our team created this website to organize a large spreadsheet of grants in a way that was more user-friendly, rather than a 20-columned document. We based our design around Embark Sustainability's branding to allow for our project to blend seamlessly when inserted into their existing website. It was also important for the website to be accessible for all users, which includes being responsive to screen widths between 320px and 1920px, in addition to being adaptable to screen-readers and keyboard navigation.

03.

Outcome

Our team's design centered around simplifying our site's navigation. For this reason, I decided to implement a breadcrumb navigation (in addition to the main header navigation) to help orient the user, as well as using an information hierarchy for users to easily search for funding opportunities that are relevant to them. Due to the amount of content on the site, I coded a split coloured background to help the user identify the main interactive elements, which almost always land on the beige side. More than anything, this project tested our adaptability, especially since we had to create solutions without JavaScript.

The final design is cohesive and uses a simplistic style to counter the amount of information on the website. The colour division is especially helpful in guiding the user towards the most important information, like the grant cards and the individual grant requirements. Our code exclusively uses rem units to ensure that the website is adaptive to almost all screen sizes and users' text size preferences. The consistency in our site makes it pleasant for users to navigate and easy for them to predict where information will appear on each page.

A few explored colour variations of the Community Grants page.

04.

Takeaways

This project taught me a lot about accessibility within web development, and the importance of designing with a varied audience in mind. Testing was our most crucial task for this project, which included testing screen readers, keyboard navigation, responsiveness, and even how the site operated on different web browsers.
